# 🎓 Smart College Campus

A full-stack Smart College Campus Management System designed to simplify college administration and improve communication between students, faculty, and administrators.

## 📖 Project Overview

This project provides a digital platform for managing various campus activities such as student records, faculty management, attendance, notices, events, and campus services.

## ✨ Features

- Student Management
- Faculty Management
- User Authentication
- Attendance Management
- Notice Board
- Event Management
- Campus Services
- Responsive User Interface

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Aishwarya-Godre/Smart-Collage-Campus.git
```

```bash
npm install
```

### Start the Project

```bash
npm start
```
